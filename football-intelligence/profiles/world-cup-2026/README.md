# World Cup 2026 Profile Database

Purpose: store competition-specific team, coach, and player intelligence for FIFA World Cup 2026.

This folder focuses on durable mechanisms, not one-off reactions.

Use these files together:

- `team-index.md` — group-by-group team map and profile status
- `team-profiles.md` — team mechanisms and recent match lessons
- `coach-watchlist.md` — manager behavior notes and verification status
- `player-watchlist.md` — player role, movement, creator/finisher notes
- `profile-update-rules.md` — when to update team/player/coach memory

Important rule: mark unknowns as unknown. Do not invent coaches, roles, heatmaps, or player tendencies without evidence.
