# Manager Profiles — Coach Behavior Model

Purpose: capture repeatable coach behavior under different match states. This file prevents the model from treating teams as static ratings and forces analysis to ask how the coach is likely to manage risk, tempo, substitutions, and tactical shape.

---

## Core Principle

A manager profile is not a personality description. It is a state-based behavior model.

For every coach, evaluate:

- Behavior when level
- Behavior when leading
- Behavior when trailing
- Behavior in must-win contexts
- Behavior when a draw is useful
- Substitution timing
- Risk appetite
- Preferred control method
- Known blind spots

Never assume:

- Draw useful = team parks the bus
- Must win = team attacks recklessly
- Big team = automatically high tempo
- Defensive coach = no attacking intent

Coach behavior must be inferred from evidence: recent matches, tactical history, quotes, lineup choices, and in-game adjustments.

## State-Based Coach Rules

### Rule M1 — Useful Draw Does Not Equal Passive Football

A useful draw lowers reckless risk, but it does not erase the desire to win. Big teams often keep pursuing victory because winning improves qualification security, public pressure, goal difference, and psychological momentum.

Apply only a small risk-reduction modifier unless there is evidence of defensive behavior.

### Rule M2 — Lead Protection Varies by Coach

Some coaches defend territory after leading; others keep the ball; others continue pressing. Identify the specific control method:

- Deep protection
- Possession control
- Counter-attacking trap
- Continued high press
- Substitution-based lockdown

Each control method affects goals, corners, and cards differently.

### Rule M3 — Trailing Does Not Automatically Mean Better Scoring Chance

Trailing usually raises volume, corners, and urgency. It does not always raise goal conversion. The opponent's defensive quality and the trailing team's chance quality must be checked separately.

### Rule M4 — Substitution Timing Is a Predictive Signal

Track whether the coach changes early or waits. Early attacking substitutions raise late-game volatility. Late conservative substitutions can preserve pre-existing low-event patterns.

### Rule M5 — Reputation Pressure Changes Behavior

Large teams under criticism may start faster and force attacks, but this can create rushed shots and transition risk. Do not count reputation pressure as purely positive.
